# LAST TRANSMISSION

A cinematic sci-fi survival horror game built with Python and CustomTkinter.

## FEATURES

- Survival resource management
- Crew psychological events
- Hidden alien threat system
- AI instability mechanics
- Dynamic terminal interface
- Random exploration events

## INSTALLATION

Install dependency:

```bash
pip install customtkinter
```

Run:

```bash
python CIP26.py
```
